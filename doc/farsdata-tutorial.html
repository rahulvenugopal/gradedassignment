This package is a submission towards final assessment of Building an R package course. This has two functions to obtain a summary and a visual map of count of fatal injuries in motor vehicle traffic crashes from 2013,2014, and 2015.

Data

Three data files were provided by the course instructors. These are from the National Highway Traffic Safety Administration (NHTSA) Fatality Analysis Reporting System (FARS) data.

You may download the full data set from this link

filename <- make_filename('2015')
data_2015 <- fars_read(filename)
